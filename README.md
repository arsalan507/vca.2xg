# Viral Content Analyzer

A standalone tool for analyzing viral content and creating replication strategies.

## Features
- Viral content analysis workflow
- Voice note recording
- Emotion & outcome tracking
- Script writer authentication
- Supabase backend

## Tech Stack
- Frontend: React + TypeScript + Vite
- Backend: Supabase (Database + Auth)
- Deployment: Vercel (Frontend) + Supabase (Backend)
